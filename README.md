# css sprite

css雪碧图简单制作工具

可以通过图片，直接生成sprite文件，并且生成代码

可以通过鼠标点击调整图片位置


#使用步骤
####1.切好需要的图片
    
####2.把切好的图放进你的项目文件夹

####3.打开CssSprite.exe运行程序，点击左上角按钮选择多涨图片

####4.自定义排版或者选择默认排版方式

####5.点击“生成雪碧图”按钮，选择目录，默认选择的是你打开图片的目录
